About

FretScale is a visualisation of a guitar's (or a guitar-like instrument's) fretboard. It aims to help learners to identify shapes based on a given root pitch and musical scale/mode.

Tuning: select the each string pitch in the form in the right of the page. Click the [x] button to remove a string or [Add String] to add a string. Click the [Set Tuning] button when done.

Key Settings: select a root pitch from the drop down in the form at the middle of the page. Then select a scale or mode for the chosen root from the other drop down. Click the [Set Fretboard Display] button when done.
